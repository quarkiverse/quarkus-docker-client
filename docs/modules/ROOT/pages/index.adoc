== Installation

If you want to use this extension, you need to add the `io.quarkiverse.docker-client:quarkus-docker-client` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>io.quarkiverse.docker-client</groupId>
    <artifactId>quarkus-docker-client</artifactId>
    <version>{project-version}</version>
</dependency>
----
